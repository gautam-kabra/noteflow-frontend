# 🎨 NoteFlow Frontend

A premium, high-performance note-taking interface built with **Next.js 15** and **Tailwind CSS**.

---

## ✨ Design Features

- 💎 **Bento Dashboard**: A modern, organized grid layout for effortless note management.
- 🎭 **Micro-animations**: Fluid transitions and interactive elements powered by **Framer Motion**.
- 🌑 **Premium Dark Mode**: A sophisticated dark aesthetic designed for focus and clarity.
- 📱 **Mobile First**: Fully responsive architecture that looks stunning on any device.
- ⚡ **Optimized Performance**: Leverages Next.js 15 App Router for blazing-fast navigation.

---

## 🚀 Deployment (Render.com)

1.  **Build Command**: `npm install && npm run build`
2.  **Start Command**: `npm start`
3.  **Environment Variables**:
    - `NEXT_PUBLIC_API_URL`: The URL of your NoteFlow backend service.

---

## 💻 Local Development

1.  **Clone the repo and navigate to frontend**:
    ```bash
    cd frontend
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Setup environment**:
    Create a `.env.local` file:
    ```bash
    NEXT_PUBLIC_API_URL=http://localhost:8000
    ```
4.  **Launch the dev server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:3000`.

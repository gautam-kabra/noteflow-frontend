# 🎨 NoteFlow Frontend

A premium, high-performance note-taking interface built with **Next.js 15** and **Tailwind CSS**.

## 🔗 Live Application
- **Live URL**: [https://noteflow-frontend-7vk7.onrender.com/](https://noteflow-frontend-7vk7.onrender.com/)

---

## ✨ Design Features
- 💎 **Bento Dashboard**: Modern, organized grid layout.
- 🎭 **Micro-animations**: Smooth transitions with Framer Motion.
- 📱 **Fully Responsive**: Optimized for mobile and desktop.
- ⚡ **Standalone Build**: Optimized production deployment.

---

## 🚀 Deployment (Render.com)
1. **Root Directory**: `frontend`
2. **Build Command**: `npm install && npm run build`
3. **Start Command**: `npm start`
4. **Env Vars**:
   - `NEXT_PUBLIC_API_URL`: `https://noteflow-backend-b4m3.onrender.com`

---

## 💻 Local Development
1. `npm install`
2. `npm run dev` (Runs on `http://localhost:3000`)
